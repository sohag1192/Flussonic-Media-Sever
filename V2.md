## üöÄ Installation

### 1\. One-Line Automatic Install

The standard method for fresh servers.

```bash
curl -sSf [https://flussonic.com/public/install.sh](https://flussonic.com/public/install.sh) | sh
```

### 2\. Manual Repository Setup (Ubuntu/Debian)

For granular control over the installation process.

```bash
wget -q -O - [https://flussonic.com/doc/install/binary/gpg.key](https://flussonic.com/doc/install/binary/gpg.key) | apt-key add -
echo "deb [https://flussonic.com/doc/install](https://flussonic.com/doc/install) binary/" > /etc/apt/sources.list.d/flussonic.list
apt-get update && apt-get -y install flussonic flussonic-ffmpeg flussonic-python
```

## üîß Performance Tuning (ISP Grade)

For high-load servers (10Gbps+), apply these kernel optimizations in `/etc/sysctl.conf`:

```ini
# Network Tuning for High Concurrency
net.core.rmem_max = 16777216
net.core.wmem_max = 16777216
net.ipv4.tcp_rmem = 4096 87380 16777216
net.ipv4.tcp_wmem = 4096 65536 16777216
net.ipv4.tcp_congestion_control = htcp
net.core.netdev_max_backlog = 30000
net.ipv4.tcp_max_syn_backlog = 4096
```

*Apply changes with: `sysctl -p`*

## üõ† Management & Troubleshooting

### Essential Commands

  * **Start:** `service flussonic start`
  * **Restart:** `service flussonic restart`
  * **Check Logs:** `tail -f /var/log/flussonic/flussonic.log`
  * **Validate Config:** `/opt/flussonic/bin/validate_conf`

### Common Issues

  * **License Locked:** If you see "License locked", contact support to reset HWID.
  * **404 Stream Not Found:** Check if the `url` source is reachable via `curl` or `ffprobe`.
  * **High CPU:** Ensure `transcoder` is using `hw=nvenc` (NVIDIA) or `hw=qsv` (Intel) if hardware is available.
